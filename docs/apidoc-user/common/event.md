# 公共事件类型

| 枚举值           | 说明        |
| ---------------- | ----------- |
| PROJECT_CREATED  | 项目创建    |
| REPO_CREATED     | 仓库创建    |
| REPO_UPDATED     | 仓库更新    |
| NODE_CREATED     | 节点创建    |
| NODE_DOWNLOADED  | 节点下载    |
| NODE_RENAMED     | 节点重命名  |
| NODE_MOVED       | 节点移动    |
| NODE_COPIED      | 节点复制    |
| NODE_DELETED     | 节点删除    |
| METADATA_DELETED | 元数据删除  |
| METADATA_SAVED   | 元数据保存  |
| VERSION_CREATED  | 版本创建    |
